The source code to Rinse is on GitHub. (MIT license)

The container is based on Alpine Linux. (MIT license)

The container uses gVisor to sandbox it's operations. (Apache 2.0 license)

Apache Tika is used for metadata extraction and language detection. (Apache 2.0 license)

LibreOffice is used to convert documents to PDF. (Mozilla Public License Version 2.0 license)

pdftoppm is used to render PDF's. (GPLv2 license)

Tesseract is used to OCR and create the rinsed PDF. (Apache 2.0 license)
